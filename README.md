# Duel[terminal]

Jogo de terminal com duelo de turnos. O duelo consiste em um chefe contra um monstro aliado.

## Funcionalidades

* Escolhe Chefe
* Escolhe Monstro
* Duelo

## Tipo Aplicação

* Jogo na linguagem Ruby

## Stack

* Ruby

## Rodando a aplicação

Dentro do diretorio execute o seguinte comando:
````bash
ruby jogar.rb
````
